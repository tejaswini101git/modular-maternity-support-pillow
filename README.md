
# Modular Maternity Support Pillow
![Modular Maternity Support Pillow Design](images/pillow_design_sketch.png.jpg)
A patent-backed ergonomic maternity support system designed to improve comfort during pregnancy and provide adaptable support during postnatal recovery.

---

## Overview

Pregnancy introduces significant biomechanical and postural changes that often lead to discomfort, musculoskeletal strain, and sleep disturbances. Traditional maternity pillows are typically rigid in design and lack adaptability.

This project proposes a modular maternity support pillow system that provides adjustable ergonomic support for the abdomen, back, hips, and legs.

The design allows users to configure the pillow according to their body posture, trimester stage, and sleeping position.

---

## Key Features

• Modular detachable pillow components  
• Ergonomic abdominal and lumbar support  
• Adjustable configuration for different sleeping positions  
• Postnatal usability for feeding and recovery support
